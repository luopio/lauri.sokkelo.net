Pet Hotel


bigger image

Written around the first years of the 21st century, Pet Hotel is a client management system written in Python. It uses MySQL as the database backend and provides an easy-to-use GTK-frontend for management. It has a "double-layered-MVC-architecture" which means that it can support multiple clients easily.

I've released it under the GPL and the application - including the source - is available here.
